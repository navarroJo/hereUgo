hereUgo

small, simple, easy to use

hereUgo is a simple, light weight and easy to use modal jQuery plug-in. It was originally built to display completed web design projects with photos of the site, information about the project and what when into completing it, but it works as an everything modal as well. No on page JavaScript required, all you have to do is add the files and markup.

Step 1: Attach hereUgo files and jQuery library

The hereUgo zip contains a combined hereUgo.slimscroll.min.js file and separate hereUgo slimmscroll files. If you don't want the scroll function or know you'll never need it, just add the stand alone hereUgo.min.js file.

Step 2: hereUgo Handler and Markup

Base Markup

Set the "data-display" attribute on th handler element equal to the ID of the hereUgo you want to display when the element is clicked.

The hereUgo Handler can be any HTML element but an anchor tag is always best.

This is the HTML markup for the hereUgo project layout.

Step 3: Set any options if you want to change the defaults

To set custom options on a hereUgo add "data-" followed by the option you want to change and equal to the new value.

The available options and values are:

data-animation [default:fade]

Sets the entrance animation for the hereUgo

  • blind
  • bounce
  • clip
  • drop
  • explode
  • fade
  • fold
  • highlight
  • puff
  • pulsate
  • scale
  • shake
  • size
  • slide
  • transfer

data-animationspeed [default:200]

Sets the speed of entrance and exit animations, can be set to any numerical value

data-closeBGclick [default:true]

Sets whether the hereUgo should close when the background is clicked or not, set to "true" or "false"
